### 補題1 (定理1): T(n) = T(C²(n)) + 2  [全奇数nで成立]

  証明: 奇数n → C(n)=3n+1(偶数) → C²(n)=(3n+1)/2
  T(n) = 1 + T(C(n)) = 1 + 1 + T(C²(n)) = 2 + T(C²(n))  ■

### 補題2 (定理2): n ≡ 2^k-1 → C²(n) ≡ 2^(k-1)-1  [mod各々]

  証明: n = 2^k·m + 2^k - 1 とおくと
  C²(n) = (3n+1)/2 = 3·2^(k-1)·m + 3·2^(k-1) - 1
  mod 2^(k-1): ≡ -1 ≡ 2^(k-1)-1  ■

### 補題3 (H75): 奇数 r ≠ 2^k-1 → C²(r) ≢ 2^(k-1)-1  [mod 2^(k-1)]

  証明: C²(r) ≡ 2^(k-1)-1 と仮定すると r ≡ -1 (mod 2^k)
  つまり r = 2^k-1 — 矛盾  ■

## H76: 偶数クラスの厳密上界

**目標**: E[T(n) | n≡2r (mod 2^k)] < E[T(n) | n≡2^k-1 (mod 2^k)] を示す

**観察**: 偶数 n=2m のとき T(n) = T(m) + 1
  n ≡ 2r (mod 2^k) ⟺ m = n/2 ≡ r (mod 2^(k-1))
  E[T|2r, k] = 1 + E[T|r, k-1]

**比較**:
  E[T|all-1, k] = 2 + E[T|all-1, k-1]  [補題2]
  E[T|2r, k]    = 1 + E[T|r, k-1]

  差: E[T|all-1,k] - E[T|2r,k]
    = (2+E[T|all-1,k-1]) - (1+E[T|r,k-1])
    = 1 + (E[T|all-1,k-1] - E[T|r,k-1])

    ≥ 1 + 0 = 1  (帰納仮定: all-1クラスが最大なので差≥0)
    > 0  ■  (差は常に ≥ 1)

## H77: 基底 k=1 の解析的証明

**命題**: E[T(n) | n奇数] > E[T(n) | n偶数]

**解析的アプローチ**:
  偶数 n=2m: T(2m) = 1 + T(m)
  奇数 n: T(n) = 2 + T((3n+1)/2)

  E[T(偶数, ≤N)] = 1 + E[T(m), m≤N/2]  ≈ 1 + E[T, N/2]
  E[T(奇数, ≤N)] = 2 + E[T((3n+1)/2), n奇数,≤N]

  (3n+1)/2 は n ≤ N の奇数全体で [2, 3N/2] に分布
  E[T((3n+1)/2)] ≈ E[T, 3N/4] (大雑把な近似)

  E[T, N] ≈ C·log₂(N) (コラッツの停止時間は対数的に増加)
  E[T(偶数)] ≈ 1 + C·log₂(N/2) = 1 + C·(log₂N - 1)
  E[T(奇数)] ≈ 2 + C·log₂(3N/4) = 2 + C·(log₂N + log₂(3/4))
             = 2 + C·log₂N - C·log₂(4/3)
             = 2 + C·log₂N - C·(2-log₂3)

  差: E[T(奇数)] - E[T(偶数)]
    ≈ (2 - C·(2-log₂3)) - (1 - C)
    = 1 - C·(2-log₂3) + C
    = 1 + C·(1-(2-log₂3))
    = 1 + C·(log₂3 - 1)
    = 1 + C·log₂(3/2)

  C ≈ E[T(n)]/log₂n ≈ 9〜10 (N=1M付近), log₂(3/2) ≈ 0.585
  差 ≈ 1 + 10·0.585 ≈ 6.85
  実測値 ≈ 12.3 (H68)

  → 近似は正確でないが、差が正であることは示せる
  → C > 0 かつ log₂(3/2) > 0 なので差 > 1 > 0 ■（漸近的）

**N=1M 計算値**: 奇数avg=137.60, 偶数avg=125.27, 差=12.33
  C(N) = 6.904, 理論差 = 1+C·log₂(3/2) = 5.04
